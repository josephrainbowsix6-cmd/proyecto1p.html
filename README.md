## 🐾 Descripción

Este proyecto muestra una **tabla de animales** con imágenes, nombres y descripciones.  
Incluye un diseño moderno con colores, sombras y efectos visuales.

---

## ✨ Características

- 🎨 Diseño moderno con **degradados**
- 🖼️ Imágenes de alta calidad
- 📊 Tabla organizada y limpia
- ⚡ Efecto *hover* interactivo
- 💻 Hecho con HTML y CSS
